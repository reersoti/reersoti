# Hi, I'm Yaroslav

I’m a 4th-year student at Lomonosov Moscow State University, studying Applied Mathematics and Computer Science.

This GitHub profile contains projects I’ve worked on during my studies, coursework, and independent practice. Most repositories are educational or experimental projects focused on algorithms, systems programming, simulations, automation, and machine learning.

## Main Areas

- **C++ systems programming**: memory management, custom containers, file abstractions, CMake-based projects.
- **Algorithms and benchmarking**: substring search algorithms, performance comparison, generated test data, visualization.
- **Python simulations**: cellular automata, graph-based propagation models, GUI-based educational tools.
- **Machine learning**: PyTorch computer vision experiments, transfer learning, model comparison.
- **Automation tools**: desktop reporting workflows, CSV processing, Excel report generation, user-facing validation.
