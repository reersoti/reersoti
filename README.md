# Hi, I'm Yaroslav

I’m a 4th-year student at Lomonosov Moscow State University, studying Applied Mathematics and Computer Science.

I use this GitHub profile as a portfolio of educational, coursework, experimental, and practice-oriented projects. My main focus is on algorithms, C++ systems programming, Python simulations, automation tools, and machine learning experiments.

---

## Main Areas

- **C++ systems programming**: memory management, custom containers, file abstractions, CMake-based projects, CTest, GitHub Actions.
- **Algorithms and benchmarking**: substring search algorithms, performance comparison, generated test data, benchmark visualization.
- **Python simulations**: cellular automata, graph-based propagation models, Tkinter-based educational tools, unit-tested simulation logic.
- **Machine learning**: PyTorch computer vision experiments, transfer learning, model comparison, notebook-based research workflow.
- **Automation tools**: desktop reporting workflows, CSV processing, Excel generation, validation, user-facing error handling.
